== Description ==

This is a simple plugin to add a widget to your website that loads random post(s) for you. :)
== Installation ==

1. Upload `random-post-loader` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add `Random Post` to your sidebar,footer and ext. from your widgets options.
